# Тестовое задание для Slotegrator

### Запуск тестов:

#### Все тесты:
./gradlew test

#### API тесты:
./gradlew api

#### UI тесты:
./gradlew cucumber

### При написании тестов были использованы следующие технологии:
- Java 11
- JUnit 5
- Rest-Assured
- Selenium WebDriver
- Cucumber
- Gradle
- Owner
- Lombok
